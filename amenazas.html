Amenazas Comunes en la Red

Conoce los ataques más frecuentes y cómo identificarlos.

Phishing

Suplantación de identidad para robar datos. ¡Haz clic para saber más!

Ver Más

Los atacantes envían correos electrónicos fraudulentos para engañar a las personas y que revelen información confidencial como contraseñas o números de tarjetas de crédito.

Prevención: Verifica el remitente y no hagas clic en enlaces sospechosos.

Ransomware

Secuestro de datos a cambio de un rescate. ¡Haz clic para saber más!

Ver Más

Un tipo de software malicioso que bloquea el acceso a tus archivos y te exige un pago para liberarlos.

Prevención: Haz copias de seguridad de tus datos y usa un buen antivirus.

Malware

Software dañino que infecta tu dispositivo. ¡Haz clic para saber más!

Ver Más

Término general para cualquier software malicioso como virus, troyanos o spyware diseñados para dañar o acceder a tu sistema.

Prevención: Instala software de fuentes confiables y mantén tus programas actualizados.
